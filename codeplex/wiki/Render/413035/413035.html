My new package contains a solution for sending StreamInsight events to a Windows Form application.

Data is randomly generated by a custom "call center" adapter and lets you explore point event queries using both a Console output and a custom Web adapter.  The Web adapter sends data to a WCF service hosted within a Windows Form application.  The Form then draws the results of the "running totals" query in a Microsoft Chart Control.

For more information abou tthe projects in this solutions, please visit my blog post on this release.
